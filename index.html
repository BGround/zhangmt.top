Nautil 2019

Nautil 是一个响应式前端开发框架，基于 React 和观察者模式而研发的新一代跨端开发集成框架。

它内置的观察者模式方案，有效解决应用的状态、数据、路由、国际化等问题，并内置这些基础部件。它的跨端方案，可以让开发者写一份代码，就可以在不同的平台上编译后运行。

Componer 2016-2017

Componer 是一个基于 node、gulp、webpack 的开源前端开发工程化工具。

前端开发已经是构建先行的时代，componer 是一个专注解决 components 的构建的命令行工具。它基于 gulp、webpack，提供 add、build、preview、test 等任务，让你完全书写 ES6 语法，通过 express 创建本地开发服务器环境，通过 karma、jasmine 框架进行测试。

WP2PCS 2014

WP2PCS 是我开发的一个基于个人云存储的网站数据备份 WordPress 插件，以百度云盘作为云存储，将网站的数据、文件定时备份到云端，为中小站长解决网站数据丢失烦恼。

它主要解决了网站数据备份的问题，因为很多网站经常出现宕机、转移、数据丢失等问题，甚至主机商跑路，苦心经营的网站瞬间化为乌有。使用 WP2PCS 可以备份 WordPress 的数据库 sql 文件，网站规定目录下的文件，到百度云盘里面，定时备份，无需手动操作，这样就解除了中小站长网站数据备份的问题。

但由于百度 PCS API 已经下线，所以现在插件也不可用了。
